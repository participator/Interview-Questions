Boxes Go 'Round

Given these three boxes, implement the necessary logic to allow the user to move the red box by clicking on the left/right buttons. Bear in mind that, in the future, more boxes could be added and that doing so should not affect the logic of the solution.

  • If the red box is in the first position, clicking “Left” should move it to the last position.
  • If the red box is in the last position, clicking “Right” should move it to the first position.
